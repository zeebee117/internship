# Dr. Maya Reynolds Therapy Website

This repository contains the code for the **Dr. Maya Reynolds Therapy Website**, a modern, responsive, and professional website for a clinical psychologist.  

The project leverages **HTML, Tailwind CSS, and vanilla JS**, optimized for both desktop and mobile users. It includes custom sections like **Our Office**, **Specialties**, **FAQs**, and a client-focused footer.
